Flashcards

Learn and master any topic with spaced repetition.

Getting Started

  1. Star subjects, decks, or chapters to set your learning scope.
  2. Use Review to strengthen previously learned material before it fades.
  3. Use Learn new to introduce a manageable batch of new cards.
  4. Use to review any subject, deck, or chapter individually.

Pro tip: Consistency beats intensity. Review daily for stronger recall

Creating a deck

  1. Create a GitHub repository containing a flashcards/ directory.
  2. Use lowercase kebab-case names for repositories and subjects, such as computer-science or classical-mechanics.
  3. Add Markdown card files inside flashcards/. Use numbered filenames such as 01_kinematics.md to control chapter order.
  4. Follow the card-writing standard for supported formats and effective SRS principles.
  5. Add the repository here, then star the material you want to learn.
Install Flashcards for faster access, resilient sessions, and reminders.

Study settings

The target is a guide, not a lock. Extra learning stays in finite batches so future review load remains visible.

Reminders use this device’s timezone and are sent only when cards are due or a session is paused.

Install on iPhone

  1. Open this page in Safari.
  2. Tap the Share button □↑.
  3. Choose Add to Home Screen, then tap Add.
  4. Open Flashcards from its new Home Screen icon and sign in.
  5. Open Settings and turn on your daily reminder.

Notifications on iPhone require the installed Home Screen app.
