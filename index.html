An in-browser spaced-repetition system that syncs to your GitHub account. Review what's important to you and outsmart the forgetting curve.

Creating Flashcards

Create a GitHub repository with markdown files in a flashcards/ folder using a Q/A or C format:

Q: What is the capital of France?
A: Paris.

C: [Paris] is the capital of France.

How-to-Use

  1. Login with GitHub and add a repository
  2. Click to start reviewing
  3. Rate your recall
  4. Come back daily to review automatically scheduled cards

Pro tip: Consistency beats intensity. 10 minutes daily is better than 2 hours weekly.
